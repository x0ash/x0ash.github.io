Arena


Overview

Arena (working-title) is a first-person arena shooter I am working on as a hobby project. Inspired by games such as Quake III Arena (1999), Unreal Tournament (1999), and Half-Life (1998), it features fast-paced combat focused around 'bunnyhopping' - a movement technique to gain additional speed.

Due to it being very early in development, the source code for Arena is currently unavailable publicly, though I have plans to open-source the project once finished.

Features

  • Client made using Unity, a widely used game engine
  • Custom server executable featuring prediction & rollback
  • Artificial intelligence for non-player characters to simulate real opponents
