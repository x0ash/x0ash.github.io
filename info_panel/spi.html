Steam Player Investigator


Overview

Steam Player Investigator is a utility created using .NET MAUI that interacts with Steam to determine if a player is a 'smurf' or not.

A smurf is a player who plays on a lesser-used account, sometimes created specifically for smurfing, in order to play against lesser-skilled players in multiplayer games. As opponents these days tend to be matched based on skill level over time, competitive integrity is lost when a player is found to be smurfing.

The Steam Player Investigator aims to conclude if a player is a smurf so users can accurately report offenders.

Features

  • Cross-platform user interface using .NET MAUI
  • Interaction with the Steam Web API
  • Webpage scraping & parsing for additional data
  • XML parsing for gathering additional data
  • Binary-classification model using account data as parameters

Additional information

Created as a project for my second-year university studies, Steam Player Investigator was created alongside a team of 5 fellow students. Throughout the six months of development we had on the project, we designed everything in the utility in consulation with each other, from the data we needed and how to obtain it to presenting the final decision to the user in the GUI.

The project was developed following a RAD pipeline with individual work coming from implementating our design work. Additionally, paired programming would be used when bridging the gap between two unique implementations.
